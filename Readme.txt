1. Description:

1.1. Cette application Web est conçue et construite pour rechercher, acheter, louer et vendre une propriété dans le monde entier.
Il utilise une interface très simple et minimaliste pour permettre aux nouveaux utilisateurs de naviguer facilement dans l'application.
Il est conçu pour effectuer les tâches nécessaires en un minimum de clics.
En plus de cela, la plupart des fonctionnalités sont dans la même page pour rendre les utilisateurs à l'aise qu'ils filtrent
dans les propriétés, sélectionnez une langue ou planifiez une visite pour afficher une propriété. Une fois l'URL de l'application Web entrée,
la page principale / de destination apparaîtra. La page de destination contient toutes les fonctionnalités et les boutons nécessaires
pour pousser démarrer une grande expérience utilisateur et le fonctionnement de l'application.

2. Installation:
2.1 Cette application Web ne nécessite pas d'installation.

3. Fonctionnement:
3.1. Pour accéder à la page Web, saisissez l'URL de tout navigateur Web prenant en charge HTML 4 et supérieur.
3.2. Pour accéder à un compte (le cas échéant), sélectionnez le bouton "Connexion" et entrez les informations de connexion, c'est-à-dire le nom d'utilisateur et le mot de passe.
3.3. Pour vous inscrire à un nouveau compte, sélectionnez le bouton "S'inscrire" et entrez les informations requises.
3.4. Pour réinitialiser le nom d'utilisateur / mot de passe, sélectionnez le bouton "Nom d'utilisateur / mot de passe oublié" dans la fenêtre contextuelle de connexion et saisissez l'e-mail.
3.5. Pour rechercher une propriété, saisissez les informations de localisation dans le champ de recherche et sélectionnez le bouton "Rechercher".
3.6. Pour acheter ou louer une propriété, cliquez sur le bouton "Acheter / Louer" et entrez les informations requises.
3.7. Pour afficher une propriété ou planifier une visite, sélectionnez la propriété souhaitée et cliquez sur le bouton "Planifier", saisissez le jour et l'heure souhaités.
3.8. Pour vendre une propriété, après vous être connecté à un compte, sélectionnez le bouton "Vendre" et créez une annonce. Saisissez les informations requises et téléchargez les fichiers comme vous le souhaitez.
3.9. Pour utiliser les filtres, vérifiez les filtres souhaités avant de lancer une recherche.
3.10. Pour changer la langue, sélectionnez la langue dans le champ de sélection de la langue en haut à droite de la page principale.
3.11. Pour agrandir les photos des propriétés, cliquez sur et sélectionnez la photo souhaitée.

4. Construit avec:
4.1. .NET framework - framework logiciel développé par Microsoft.
4.2. Microsoft Visual Studio 2019 - un environnement de développement intégré de Microsoft.
4.3. C # - langage de programmation polyvalent multi-paradigmes.

5. Bogues connus et fonctionnalités non prises en charge:
5.1. L'application Web ne contient aucun bogue connu.
Actuellement, les fonctionnalités non prises en charge: backend, base de données,
et les problèmes liés à ceux tels que l'enregistrement des nouvelles informations de compte et des informations de propriété.
Le site Web ne propose actuellement que 2 langues.
